FAQ’s:

   1. How to display list items side by side?

   A. By using

           a) display “inline” for <li> element, but it effects the numbering style.

           b) display “flex” for <ol> element with direction “row”, it keeps numbering side by side.



Ex:

<!DOCTYPE html>

<html lang="en">

<head>

    <meta charset="UTF-8">

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <title>Document</title>

    <style>

        .child-list  {

            display: flex;

            flex-direction: row;

            margin-top: 20px;

        }

        .child-list li {

            margin-right: 100px;

        }

    </style>

</head>

<body>

   <ol>

      <li> __________ elements doesn't require end token. 

           <ol type="a" class="child-list">

                <li>Normal</li>

                <li>Void</li>

                <li>RC</li>

                <li>Raw Text</li>

           </ol>

      </li>

   </ol>

</body>

</html>





   2. How to display in multiple columns?

   A. By using

           a) display “grid” 

           b) columns of CSS



Syntax:

        ol {

           display: grid;

           grid-template-columns: 6fr 6fr;

        }



       ol {

           columns: 2

       }

   3. How to design scrollable list?

   A. By using CSS “overflow” set to auto according to width and height. 



Syntax:

           ol {

                width: 100px;

                border: 1px solid gray;

                padding: 10px;

                overflow: auto;

                height: 100px;

           }

   4. How to remove numbering for list?

   A. By using CSS “list-style” attribute set to “none”.



Syntax:

       ol {

            list-style: none;

       }

   5. How to design “breadcrumb”? 

   A. By using CSS classes 

           a) ::before

           b) ::after

       They allow to set a symbol before or after. 



CSS Child Selectors:



       :first-child            

       :last-child

       :nth-child(n)            // n refers to level number



Syntax:

       li : nth-child(2) { }

       li : nth-child(even | odd) { }



Ex:



<!DOCTYPE html>

<html lang="en">

<head>

    <meta charset="UTF-8">

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <title>Document</title>

    <link rel="stylesheet" href="../node_modules/bootstrap-icons/font/bootstrap-icons.css">

    <style>

       ol {

         list-style: none;

         display: flex;

       }

       li::before {

          content: " > ";

          padding-left: 10px;

          padding-right: 10px;

          color:gray;

          font-size: 14px;

       }

       li:first-child::before {

          content: "";

       }

       li:nth-child(odd) {

          color:blue;

       }

    </style>

</head>

<body>

   <ol>

      <li> Home </li>

      <li> Electronics </li>

      <li> Mobiles & Accessories </li>

      <li> Mobiles </li>

      <li> Apple Mobiles </li>

      <li> iPhone </li>

   </ol>

</body>

</html>



Unordered List:

   - It sets bullet symbol for items in a list.

   

   <ul>            unordered list

   <li>            list item



Syntax:

       <ul>

         <li> Item-1 </li>

         <li> Item-2 </li>

       </ul>



   - You can set “type” as “circle, disc & square”.



       <ul type=“circle | disc | square”>



  FAQ: How to set custom image as bullet ?

  Ans: By using  “list-style-image” attribute set with “url()”.





Syntax:

       {

         list-style-image: url(“./images/pic.jpg”);

       }
