CSS Units

   - CSS units are for measuring dimensions and positions. 

   - Units are categorized into 2 groups

           1. Relative Units

           2. Absolute Units

   - Absolute units are independent and have a fixed size or position, they are not effected by

     parent or adjacent elements. 

   - All units are converted into pixels. 

   - Browser supports following absolute units. 






Unit




Name




Equivalent to






cm 




Centimeters




1cm = 96px/2.54






mm 




Millimeters




1mm = 1/10th of 1cm






Q




Quarter-millimeters




1Q = 1/40th of 1cm






in




Inches




1in = 2.54cm = 96px 






pc 




Picas




1pc = 1/6th of 1in






pt 




Points




1pt = 1/72th of 1in






px 




Pixels




1px = 1/96th of 1in














Syntax:

        {

           width: 2Q;

           height: 4in;

           margin-left: 50pt;

           margin-top: 50mm;

       }



   - Relative units depend on parent or adjacent elements.

   - CSS relative units are 



               a) em            element size

               b) rem            root element size

               c) vh                view port height

               d) vw                view port width 

               e) %                percent 



   - “em” & “rem” are used to control font size and can be used only for text.

   - If you are applying “em” & “rem” for any other context, then it uses the same font size of 

     parent and its root.



Ex:

<!DOCTYPE html>

<html lang="en">

<head>

   <meta charset="UTF-8">

   <meta name="viewport" content="width=device-width, initial-scale=1.0">

   <title>Document</title>

   <style>

      .container {

         font-size: 60px;

      }

      .container p {

         font-size: 0.5em;

      }

      .container div {

         font-size: 1.5em;

         border:1px solid black;

         padding: 5px;

         width: 1em; 

      }

      .container span {

         font-size: 1.5rem;

      }

   </style>

</head>

<body>

   Body Text

   <div class="container">

        Container Text 

        <div>Container Div</div>

        <p>Container Paragraph</p>

        <span>Container Span</span>

   </div>

</body>

</html>



   - “vw” & “vh” are relative to view port.

   - View Port refers to active screen size of device. [w-width, h-height]

   - It is not relative to parent, it is relative to screen. 



Syntax:

           img {

               width: 100vw;

               height: 100vh;

           }



   - 100 refers to 100% for screen width and height.

   - The units in “%” are not relative to screen, they are relative to parent element.

   - If parent is not defined the they are relative to browser width & height. 



Syntax:

           img {

               width: 100%;

           }



FAQ: How to configure fluid images?

Ans:  By using relative units. [%, vh, vw]
