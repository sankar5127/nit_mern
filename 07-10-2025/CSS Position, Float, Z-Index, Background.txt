CSS Positions:

   

1. Static

2. Absolute

3. Fixed

4. Relative

5. Sticky 



Static Position

   - It is the default position for element.

   - It keeps element according to the normal flow of document.

   - It can’t move by using left, right, top or bottom attributes. 



Absolute Position 

   - It removes element from normal flow of document. 

   - It allows to move element using left, right, top and bottom attributes.

   - It keeps the element with regard to content in page.

   

Fixed Position

   - It removes element from normal flow of document.

   - It allows to move element using left, right, top and bottom attributes.

     - It keeps the element with regard to browser window, not content in page.



Sticky Position

   - It keeps element according to normal flow of document.

   - It will not allow to move element using left, right, top and bottom attributes. 

   - You can stick at specific position using top, left, right and bottom.



Relative Position

   - It is the position defined for parent element.

   - It can keep the child elements position with regard to parent.

   - You have to make sure that child element position is always “absolute”.

CSS Z-Index

    - It is used to control the overlapping elements in page.

    - It allows to send element back or backward and front or forward.

    - To stack elements you have to configure position as “absolute”. 

    - All elements in stack must be configured using “z-index”. 

    - Z-index starts with “0”, which is behind all elements in stack.



Syntax:

       div {

           position: absolute;

           z-index: 0;

       }

CSS Float & Clear

   - It allows to float an element to left or right.

   - Float will keep the element floating with content in backdrop. 

   - Float effects the adjacent elements. 

   - Hence the adjacent elements requires a “clear” attribute to clear left, right or both sides.



Syntax:

   div {

        float :left;

   }

   p { 

       clear: left;

   }

   - background-clip [ border-box, content-box]
