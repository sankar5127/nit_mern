Hyperlinks in HTML



- Hyperlink is clickable text, picture or graphic that navigates user from one location to another.

- According to the web standards a website or application must have proper navigation mechanism.

- Every service website must provide a “Sitemap”.

- Hyperlinks enable good navigation in any application. 

- They are classified into 2 types

       a) Intra Document Links

       b) Inter Document Links

- Both intra and inter document links are created using “<a>” anchor element. 



Intra Document Links

  - Intra document link refers to a hyperlink that navigates user from one location to another

   within same page.

  - The target location is marked by using “id”, It can be to any element in page. 

  - You can directly invoke the ID from address bar of browser using “#” and “id” reference.



Syntax:

                   <div id=“logo”> </div>



                   http://127.0.0.1:5500/public/page.html#logo



   - You can provide a hyperlink that navigates the specified “id”. 



                   <a  href=“#logo”>  Text | Image | graphic </a>



FAQ: What is difference between “src” & “href”?

Ans:  “src” is a getter, which is used to get and use the path. 

         “href” is a setter, it sets path.



FAQ: How to remove underline for hyperlinks?

Ans:  By using CSS “text-decoration” attribute set to “none”.



Syntax:

               a {

                   text-decoration: none;

               }



FAQ: How to change colors for visited and active links?

Ans:  a) By using <body> attributes : alink & vlink.

          b) By using CSS classes : active, visited



Syntax:

           <body alink=“red”  vlink=“gray”>



Syntax:

            a : active {    }            // when mouse is down

            a : visited {   }            // when anchor is clicked

            a : hover  {   }            // when mouse is over the link

FAQ: How to highlight the target location if page content is not scrollable?

Ans:  By using CSS target class, you can identify the target and apply styles.



Syntax:

           .card { 

       

           }

           .card : target {        => card can be target when it is defined with “id” 

                

           }
