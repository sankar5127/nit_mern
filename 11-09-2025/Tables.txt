Table Elements

   <table>                    start and end of table

   <thead>                    header section

   <tbody>                    body section

   <tfoot>                    footer section

   <tr>                            table row in header, body or footer

   <td>                            table data for normal cell

   <th>                            table header for headings

   <caption>                caption to make table SEO friendly 

   <colgroup>                groups columns for configuring styles



Table Attributes:

   

1. Frame, Rules and Border



           frame                : It sets a frame for table with values 

                                     “box, rhs, lhs, above, below, void”.

                               

                                     <table frame=“box | rhs | lhs ..”>



           rules                : It sets rules for rows, cols and groups with values

                                     “all, rows, cols, groups, none”.



                                     <table frame=“box”  rules=“rows | cols | groups | all..”>



           border                : It sets border for cells. It can have only 0=false, 1=true.                                      You can’t set border if rules attribute is defined.



                                     <table border=“1” frame=“void”>



2. Cell Spacing & Padding



           cellspacing            : It sets space between cells.

           cellpadding            : It sets space between cell border and content inside cell. 



           <table cellspacing=“10” cellpadding=“20”  border=“1” frame=“void”>





3. Background color & Image 



           bgcolor                : It sets background color for table, row, group, cell.

           background        : It sets background image for table, row, group, cell.



        <table    bgcolor=“#0f0”>

       <tr bgcolor=“yellow”>

       <td bgcolor=“red”>

       <thead bgcolor=“black”>

       <table background=“./images/pic.jpg”>



4. Align & Valign

       

       align            : It aligns the cell content left, center, right or justify 

       valign            : It aligns the cell content vertically top, center, bottom.



       <tr align=“center” valign=“top”>

       <td align=“right” valign=“center”>
