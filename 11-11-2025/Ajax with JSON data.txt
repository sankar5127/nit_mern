             fetch('../data/products.json')

            .then(function(data){

                      var tr = document.createElement("tr");

                        tr.appendChild(tdTitle);

                        tr.appendChild(tdImage);

                        tr.appendChild(tdPrice);

                        tr.appendChild(tdRating);

                        document.getElementById("tblProductsBody").appendChild(tr);

                 })

            })

        }

                                                       Distributed Computing



- Distributed computing allows communication between two applications running on two different devices.



- It also allows sharing of information between two different objects running in two different process of same machine. [device]



Distributed computing technologies:

   

       CORBA            Common Object Request Broken Architecture [ 14+ languages ]

       DCOM                Distribute Component Object Model [ Visual Basic ]

       RMI                    Remote Method Invocation [ J2EE ]

       Java Beans     Java

       Remoting        .NET

       Web Service  All technologies can use [C, Java, .NET, PHP, Python etc..]

  

Communication Specifications:



       a) SOAP

       b) REST

       c) JSON



SOAP

   - Service Oriented Architecture Protocol 

   - Consumer sends an XML request.

   - Provider sends an XML response. 



   Request:

               <Movie>

                   <Genere> Thriller </Genere>

               </Movie>



   Response:

               <Movies>

                   <Movie>

                       <Title> A </Title>

                       <Director> something </Director>

                       <Genere> Thriller </Genere>

                   </Movie>

                   …

                   …

               </Movies>



REST

       - Representational State Transfer

       - Consumer sends a simple query request

       - Provider sends XML or JSON response. 



       Request:

               https://www.netflix.com/genere/thriller 

               https://www.netflix.com/movies?genere=thriller 



       Response:

               [

                   {  Title: “A”, Director: “something”, Genere: “thriller” }

                   ..

                   ..

               ]



JSON

       - JavaScript Object Notation

       - Consumer sends JSON request.

       - Provider sends JSON response. 



Issues with Web Service:    

   - It uses SOAP.

   - It can run on only HTTP or HTTPS protocols. 

   - It can run only on Web Server.

   - Response is always XML.



API

   - Application Programming Interface

   - It can use REST & JSON.

   - It can run on any protocol.

   - It can run on any server. 

   - It can run on any device or any OS service. 

   - It is light weight & faster.
