CSS Media Queries 

   - Media queries are required to make the page responsive.

   - You can design for various orientations, width and height of devices. 

   - @media is a directive to define queries. 



Syntax:

       @media  type  and  (query)

       {



       }



       type            : screen, print, speech, all



       query        : orientation, min-width, max-width, width



       orientation    : It can be landscape or portrait.



       width            : exactly specified units.



       min-width    : starting from specified upto end. 



       max-width    : starting from zero upto specified.



Ex:

       @media screen and (min-width:600px)

       {

       }

FAQ: What are the basic requirements to configure a responsive page?

Ans:         

       1. Set <meta> viewport to fit content according to device width.



               <meta name=“viewport”  content=“width=device-width, initial-scale=1.0”>



       2. Configure responsive images and container by using dimensions relative to parent.



               <img width=“100%”>

               <div style=“width=50%”> 



       3. Use Relative units for font size.



                child {

                    font-size: 0.5em;

                }



       4. Use Media queries 



               @media { }





                                                      Bootstrap 

- It is a powerful, extensible, rich featured front toolkit used to build interactive and responsive UI.

- It provides components and grid system to create responsive designs. 

- You can rapidly build interactive UI.



Setup Bootstrap for project:



1. Install following libraries



           > npm   install bootstrap --save

           > npm   install bootstrap-icons --save



    Note: The latest version of bootstrap is “5”, it doesn’t require jQuery explicitly. 

             Bootstrap versions upto 5 require jQuery. 

           

           > npm install jQuery --save         // not required for latest version



2. Link the following files to webpage



 <link rel=“stylesheet” href=“../node_modules/bootstrap-icons/font/bootstrap-icons.css”>

 <link rel=“stylesheet” href=“../node_modules/bootstrap/dist/css/bootstrap.css”>

 <script src=“../node_modules/bootstrap/dist/js/bootstrap.bundle.js”> </script>
