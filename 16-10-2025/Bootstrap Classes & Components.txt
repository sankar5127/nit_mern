Bootstrap Components

   - Component comprises of design, styles and functionality. 

   - Design with HTML.

   - Styles with CSS 

   - Functionality with JavaScript library [jQuery]



1. Alert

   - It is an embedded message box inside page. 

   - You can dynamically control the message. [open / close ] 



Classes:

   .alert

   .alert-{contextual}

   .alert-dismissible 

   .alert-title

   .alert-subtitle 

   .alert-text

   .alert-link

   

 Attribute: [ to control alert ] 



   data-bs-dismiss        : It is used for button in alert to close the alert dynamically. 

                                     It refers to component name.



Syntax:

    <div class=“alert alert-dismissible”>

        <button data-bs-dismiss=“alert”> </button>

        … your content …

   </div>



Ex:



<!DOCTYPE html>

<html lang="en">
