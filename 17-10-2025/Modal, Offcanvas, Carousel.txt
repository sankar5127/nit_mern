2. Modal 

    - It is a dialog that pops up with content in background.

    - You can control dynamically with open & close. 



Classes 

       .modal

       .modal-dialog 

       .modal-content 

       .modal-header

       .modal-body

       .modal-footer

       .modal-dialog-centered 

       .modal-fullscreen 

       .modal-dialog-scrollable 

       .fade 



Attributes:

       data-bs-target            : It refers to the “ID” of component 

       data-bs-toggle            : It refers to the component name [handles on click]

       data-bs-dismiss        : It refers to the component name [to close]



Syntax:

       <div class=“modal”>

           <div class=“modal-dialog”>

               <div class=“modal-content”>

                   … header …

                   … body …

                   … footer …

               </div>

           </div>

       </div>



Syntax: Position



           <div class=“modal-dialog  modal-dialog-centered  modal-fullscreen … >

           </div>



Note: Nested modals are disabled in bootstrap 5.

3. Offcanvas 

       - It is similar to modal with content in backdrop but sets the position left, right, top or 

         bottom of screen. 



Classes

       .offcanvas 

       .offcanvas-start

       .offcanvas-end

       .offcanvas-top

       .offcanvas-bottom

       .offcanvas-header

       .offcanvas-body

       .offcanvas-title 



Attributes

       data-bs-toggle

       data-bs-target

       data-bs-dismiss 



Syntax:

       <div class=“offcanvas offcanvas-start”>

             … header ..

             … body …

       </div>



Ex:

<!DOCTYPE html>

<html lang="en">

<head>

    <meta charset="UTF-8">

4. Carousel 

       - It is used to create content with sliding and fading effects. 

       - You can create a slideshow of content using carousel. 

       - It can have auto and manual control. 



Basic Carousel Classes

       .carousel

       .carousel-inner 

       .carousel-item 



Syntax:

       <div class=“carousel”>

           <div class=“carousel-inner”>

               <div class=“carousel-item”>                        item-1

               </div>

               … all items ..

           </div>

      </div>



   Note: Carousel can’t display items. You have to make sure that any one carousel-item is

            “active”. 

                           <div class=“carousel-item active”> </div>



           Carousel can’t start animation by default. You have to manually start by using the

           attribute “data-bs-ride”. 



                           <div class=“carousel”  data-bs-ride=“carousel”>



           Carousel requires animation style “slide” or “carousel-fade”.



                            <div class=“carousel slide”> 

                            <div class=“carousel carousel-fade”>



Ex:

<!DOCTYPE html>

<html lang="en">

<head>

    <meta charset="UTF-8">

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

     <link rel="stylesheet" href="../node_modules/bootstrap-icons/font/bootstrap-icons.css">

    <script src="../node_modules/bootstrap/dist/js/bootstrap.bundle.js"></script>

Carousel Controls

       - Controls allow to navigate between slides manually. 

       - Move to previous or move to next. 



Classes

       .carousel-control-prev

       .carousel-control-prev-icon

       .carousel-control-next

       .carousel-control-next-icon 



Attributes

       .data-bs-slide=“prev”

       .data-bs-slide=“next” 



       Note: You can set color for buttons using “carousel-dark” theme. 

                Bootstrap 5 changed the class with attribute. 

                

                <div data-bs-theme=“dark”>                           New

                <div class=“carousel carousel-dark”>            Obsolete 



Ex:

<!DOCTYPE html>

<html lang="en">

Carousel Indicators

       - Indicators allow to navigate to any specific slide. 



Class

       .carousel-indicators 



Attributes

       data-bs-slide-to=“index”    0,1,2..

       data-bs-target



   - Make sure that any one indicator is active. 



Ex:

Note: You can set timing of every slide using “data-bs-interval”.



          <div class=“carousel-item” data-bs-interval=“3000”>      3seconds
