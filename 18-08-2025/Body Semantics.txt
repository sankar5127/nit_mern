1. Header 

   - It is a semantic container used to display content at top margin of page.

   - Typically header comprises of 

               a) brand name

               b) brand logo

               c) navbar

               d) shortcuts 

               e) search bar etc.

Syntax:

       <header>

           … your content … 

       </header>



   Note: To access virtual path of project you can use the following:



                       ./            invokes current folder

                       ../            invokes folder at previous level. [one level back]

                       ../../        two levels back.



                       /            forward slash for virtual path.    [ localhost/project/images ]

                       \            back slash from physical path.  [ D:\images ]



2. Aside

       - It describes content aside from current context. 

       - It contains links that navigate user fro current website to any another partner site or 

         a social networking site. 

       - Typically advertisements and social media links are kept in aside. 





3. Nav

       - It is used to configure navigation area.

       - Nav comprises of items that navigate used to any named location in the same website.

       - It is used for primary navigation in page.

       - If you are configuring any secondary navigation in page then creating using a <div>

         and set role as “navigation”.



Syntax:

       <header>

           <nav> Primary Nav Items </nav>

           <div role=“navigation”> Secondary Nav Items </div>

       </header>



CSS Display Flex:

   - It is a CSS style used for parent element to keep all child elements flexible.

   - You can set flexible direction row or column. [ default is row ]



                   parent 

                   {

                     display: flex;

                     flex-direction: row | column ;

                   }



   - You can align elements inside parent by using the attributes

               a) justify-content

               b) align-items



   - Justify Content can keep elements horizontally and vertically aligned 

               a) baseline     [ if direction is row the its left, if direction column the it is top ]

               b) center

               c) end

               d) space-between

               e) space-around

               f) space-evenly
