5. Collapse 

   - It is used to show / hide any content dynamically. 



Classes 

       .collapse

       .show



Syntax:

       <div class=“collapse show”>  

           … your content …

       </div>

6. Accordion 

       - It is a set of collapsable content with mutex mechanism. 

       - Mutex is “Mutual Exclusion”. If one is expanded then all others will collapse.



Classes 

       .accordion

       .accordion-item

       .accordion-header

       .accordion-body

       .accordion-collapse 

       .accordion-button 



Attributes

       data-bs-target

       data-bs-toggle 

       data-bs-parent   [ It is defined for collapse in accordion to exhibit mutex ]



Syntax:

       <div class=“accordion”>

           <div class=“accordion-item”>

                 <div class=“accordion-header”>     

                       … accordion button …

                 </div>

                 <div class=“accordion-collapse collapse”>

                   <div class=“accordion-body”>

                           … body content …

                   </div>

                </div>

           </div>

            …. multiple  items …

       </div>

7. Dropdown 

       - Configures a dropdown list with various orientations.

       - List comprises of items with complex content. 



Classes:

       .dropdown

       .dropup

       .dropstart

       .dropend 

       .dropdown-menu

       .dropdown-item

       .dropdown-toggle 

       .dropdown-item-text 

       .dropdown-divider 



Attribute:

       data-bs-toggle    [ for button to toggle dropdown]



Syntax:

       <div class=“dropdown”>

           <button class=“dropdown-toggle”> Text </button>

           <ul class=“dropdown-menu”>

               <li class=“dropdown-item”> <span class=“dropdown-item-text”> Item </span> </li>

           </ul>

       </div>



Ex:

<!DOCTYPE html>

<html lang="en">

<head>

8. Nav & Tabs

       - It is used to configure a tab based navigation in page.

       - A page can display various content in multiple tabs. 

       - User can switch between tabs dynamically. 



Classes 

       .nav

       .nav-tabs

       .nav-pills 

       .nav-menu

       .nav-item

       .nav-link

       .nav-text 

       .tab-content 

       .tab-pane

       .active



Attributes:

       data-bs-target

       data-bs-toggle 



Syntax:

       <ul class=“nav nav-tabs”>

           <li  class=“nav-item”> <a  class=“nav-link”> Text </a> </li>

       </ul>

       <div class=“tab-content”>

           <div class=“tab-pane”>

               …

           </div>

           … multiple tab panes …

       </div>



Ex:

<!DOCTYPE html>

<html lang="en">
