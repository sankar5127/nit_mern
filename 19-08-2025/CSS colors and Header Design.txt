CSS Colors:

   1. Color Name

               {

                color: yellow;

               }



   2. Shade Name

               {

                color: lightyellow;

               }



   3. Hexadecimal 

               {

                 color: #rgb  (or) #rrggbb 

               }



   4. rgb()

               - It is a combination of Red, Green and Blue color.

               - Each color value can range from 0 to 255.



               {

                 color: rgb(redValue, greenValue, blueValue);

                 color: rgb(255, 0, 0);

               }



   5. rgba()

               - It is a combination of Red, Green, Blue and Alpha.

               - Alpha is transparency ranges from 0 to 1. 

       

               {

                 color: rgba(redValue, greenValue, blueValue, alphaValue);

                 color: rgba(0,0,0, 0.6);

               }



   6. hsl()

               - It is a combination of Hue, Saturation and Lightness. 

               - Hue is an angle of colors

                           0 to 120 degrees => Red

                           121 to 240 degrees => Green

                           241 to 360 degrees => Blue

               - Saturation is grey % from 0 to 100. 

                           0 => complete grey shade

                           100 => removes grey from color 

               - Lightness is brightness from 0 to 100%

           

               {

                  color: hsl (hue, sat, light);

                  color: hsl (10deg, 50%, 50%);

               }



   7. hsla()

               - It is a combination of Hue, Saturation, Lightness and Alpha.

           

               {

                 color: hsl(hue, sat, light, alpha);

                 color: hsl(10deg, 50%, 50%, 0.6);

               }



               <body bgcolor=“rgb(255,0,0)”>     // invalid



   8. Gradient Colors

               - Gradient a set of colors in specific orientation. 

               - You can apply gradient only by using “background-image” attribute of CSS.

               - Recommended gradients are

                   a) linear-gradient

                   b) radial-gradient



                {

                 background-image: linear-gradient(direction, color1 with %, color2);

               }

                                       

               direction: top, left, right, bottom, angle.

               color with 1 to 100% occupency.
