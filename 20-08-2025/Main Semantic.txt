CSS Display Grid



           display : grid                     : It configures a grid container.

           grid-template-columns   : It separates the container into multiple columns. 

                                                      You have to split columns among 1200px or 12fr [fractions]

Syntax:



           container

           {

               display : grid;

               grid-template-columns: 4fr 4fr 4fr;     // 3 columns

           }



CSS Box Shadow & Text Shadow



           box-shadow        : It sets shadow for container.

           text-shadow        : It sets shadow for text in container.



   Note: Shadow comprises of 4 values

               a) horizontal shadow [px]

               b) vertical shadow [px]

               c) blur [px]

               d) color

       

Syntax:

               container {

                       box-shadow: 0px 0px 0px color;

                       text-shadow: 0px 0px 0px color;

               }



CSS “hover” class:

       - It defines actions to perform when mouse pointer is over the element.

       - A pre-defined class is inherited by using “:” inheritance operator.



Syntax:

           div {

               // normal state;

           }

           div : hover {

               // mouse over state;

           }
