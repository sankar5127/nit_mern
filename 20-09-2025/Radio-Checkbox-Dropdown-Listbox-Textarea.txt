Radio Button

   - Radio allows to select one or multiple, however it will not allow to deselect. 

   - Hence radio is used with “Mutex” mechanism, which is “Mutual Exclusion”. 

   - Mutex can be configured by specifying same name for all radios in a category.

   

Syntax:

       <input type=“radio” name=“gender”> Male

       <input type=“radio” name=“gender”> Female 



   - Radio’s can’t display any text as caption. 

   - You have to set caption by using <label> element. 

   - Label uses “for” attribute, which is a data binding attribute. 

   - You can connect label with any element.



Syntax:

       <input type=“radio” name=“gender”> <label for=“gender”> Male </label>



   - Radio’s submit “on” as default value, hence it is recommended to define a value for

     every radio. 



Syntax:

     <input type=“radio” name=“gender” value=“Male”>  <label for=“gender”> Male </label>



   - You can keep any radio selected by using “checked” attribute.



Syntax:

    <input type=“radio” name=“gender” value=“Male” checked>

Checkbox

   - It allows to check and uncheck any option.

   - You can use for selection of one or multiple from a group of choices.



Syntax:

       <input type=“checkbox” name=“course” value=“UI” checked> 

       <label for=“course”> UI </label>

Dropdown List

   - It enables the user to select any one option from a group of choices. 

   - It can dropdown a list using <option> element. 

   

Syntax:

       <select>

            <option> Item-1 </option>

            <option> Item-2 </option>

       </select>



   - Every option is RC type element. 

   - It will not allow complex formats for text.

   - Every option comprises of 3 attributes

       a) value

       b) selected

       c) disabled

   - “value” defines the value to submit.

   - “selected” keeps the option selected every time on page load.

   - “disabled” will not allow to select the option.



Syntax:

           <option value=“some” selected> Text </option>

           <option value=“some” disabled> Text </option>

ListBox 

   - You can convert a dropdown into list box by using “size” attribute.

   - You can also enable selection of multiple options by using “multiple” attribute.



Syntax:

   <select size=“3” multiple>

       <option> 1 </option>

       <option> 2 </option>

   </select>



Note: You can’t have nested dropdown or listbox.



Textarea

   - It is used to define a multiline textbox.

   - It is RC type and can be used only for plain text content.
