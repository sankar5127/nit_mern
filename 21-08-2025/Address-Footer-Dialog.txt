CSS Child and Sibling Selectors:



       nav   span   {   }               It refers to all span elements inside nav. 

                                               <nav>

                                                    <span> </span>

                                               </nav>



       nav+span    {   }                It refers to first span element after nav. [Adjacent]



                                               <nav> </nav>

                                               <span> </span>     // only first

                                               <span> </span>



       nav~span   {   }                 It refers to all span element after nav but within the same

                                                scope.

                                               <nav> </nav>

                                               <span> </span>     ] all 

                                               <span> </span>     ] all

9. Dialog

       - A dialog is used to create popup message boxes. 

       - Dialog is hidden by default, you have to open dynamically.

       - Dialog pops up with content in backdrop. 



Syntax:

       <dialog open>

           … your content …

       </dialog>
