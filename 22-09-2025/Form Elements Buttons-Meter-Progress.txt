Meter 

 - It is used as grade meter to show the status of task processed.

 - It uses an indicator to show the amount of action performed. 

 - Status is displayed by using

       a) min

       b) max

       c) value

       d) low

       e) high

 

 - Low is a value defined above min and below value. 

 - High is a value defined above low and below value.

 - If low and high are not defined the default they are zero. 

 - If low and high are zero the color is green. 

 - If low and high difference is less then color is red.

 - If low and high difference is more then color is gold. 



Syntax:

       <meter min=“1” max=“100” value=“100” low=“40” high=“80”>  Gold

       <meter min=“1” max=“100” value=“100” low=“60” high=“80”>  Red

         <meter min=“1” max=“100” value=“100” low=“0” high=“0”>  Green

Progress 

   - It is used to show progress of any task performed in page, which includes 

     installing, copying, downloading, uploading, loading etc.

   - It is controlled dynamically using script. 

   - You can configure attributes

           a) min

           b) max

           c) value



Syntax:

       <progress min=“1” max=“100” value=“30”> 30%

Buttons

   - Buttons are used to confirm user actions.

   - Buttons are categorized into 2 types



   a) Generic Buttons

   b) Non Generic Buttons



   Generic Buttons

       a) HTML 4 Generic Button

               <input type=“submit”>    Submits form data

               <input type=“reset”>    Resets form data

       

       Syntax:        

           <input type=“submit” value=“Register”> 

           <input type=“reset” value=“Cancel”>

       

       b) HTML 5 Generic Button

             <button type=“submit”> Text | Icon | Image </button>

             <button type=“reset”> Text | Icon | Image </button>



       Note: If type is not defined then the default type is “submit”.



   Non Generic Button

       a) HTML 4

                   <input type=“button” value=“Print”>

       b) HTML 5

                   <button type=“button”> Text | Icon | Image </button>
