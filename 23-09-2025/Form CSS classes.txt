Form CSS Classes 

   - Class is a program template.

   - It provides predefined data and logic, which you can customize and implement according

     to the requirements.

   - CSS provides several built in classes for formatting form elements. 



1. Element State Classes



           :hover                : It defines the formats to apply when mouse pointer is over element.            :focus                : It sets formats when element gets focus. 

           :read-only            : It sets formats when read-only is defined. 

           :disabled            : It defines formats on disable state.

           :checked            : It sets formats when radio or check box are checked. 

           :visited                : It sets formats for visited links

           :active                : It sets formats when mouse pointer is down on element. 

           :target                : It sets formats when element is target of any hyperlink.

<!DOCTYPE html>

<html lang="en">

<head>

    <meta charset="UTF-8">

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <title>Document</title>

    <style>

       input:read-only {

          cursor: not-allowed;

          background-color: rgb(231, 229, 229);

       }

       input+div {

          display: none;

       }

       input:read-only+div {

         display: block;

         color:red;

         font-size: 10px;

       }

       button:disabled {

          cursor: not-allowed;

       }

    </style>

</head>

<body>

    <h2>Register</h2>

    <div>

         <label>User Name</label>

         <div>

            <input type="text" class="UserName"  value="John" >

            <div>You can't modify User Name</div>

         </div>

    </div>

    <div>

         <label>Mobile</label>

         <div>

            <input type="text" class="Mobile"  readonly  value="+919876543210">

            <div>You can't modify mobile.</div>

         </div>

    </div>

    <div>

        <button disabled> Register </button>

    </div>

</body>

</html>
