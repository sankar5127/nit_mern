Note: “required” class can’t validate required. It defines effects only when required is 

         specified for any element.



        To verify the validation you have to implement “valid” class. 



Syntax:

           input: required {

                                               => if required is defined for element

           }

           input : valid {

                                               => if required is satisfied 

           }

                                                          Multimedia 

- Multimedia refers to animation, audio and video. 

- HTML provides various elements for handling multimedia.



       a) Marquee

       b) Video

       c) Audio 

       d) Embed 



Marquee 

   - It sets scrolling and sliding content in page.

   - It is deprecated and no longer recommended for modern web.



Syntax:

       <marquee> your content </marquee>



Attributes:

       1. scrollamount            It controls the marquee speed from 1 to 100

       2. direction                It sets scrolling direction left, right, up & down. 

       3. behavior                It sets animation behavior as “alternate”.

       4. loop                        It sets loop count. The number of times to display after load.

       5. bgcolor                    Sets background color.

       6. width    / height            It sets width  & height for marquee content



Syntax:

   <marquee scrollamount=“20”  loop=“3” direction=“right” bgcolor=“yellow”>

       .. your content …

   </marquee>



Syntax:

   <marquee behavior=“alternate”>  your content </marquee>
