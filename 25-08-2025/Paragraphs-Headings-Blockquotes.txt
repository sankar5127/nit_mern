11.  Headings

   - The element <hn> is used for headings.

   - “n” refers to level number from 1 to 6.



Syntax:

       <h1> Heading 1 </h1>

        …

        …

       <h6> Heading 6 </h6>



FAQ’s:

1. Why you need a heading element if the same style can be defined without element?

A. To keep topics SEO friendly. 

    Web crawlers can make a summary of topics in page by accessing heading styles.



2. What are the styles applied for heading element?

A.    Default styles are:

       font-size

       font-weight

       display: block



3. How to remove any default style?

A. By using CSS inheritance value “unset”.



Syntax:

           h1 {                                                      h1 {

                 font-weight: unset;

                 font-size: unset;                                all : unset;

                 display: unset;                               

           }                                                            }





4. Can you change the default style?

A. Yes.





Note: 

       a) Avoid using too many headings in a page. 

       b) Don’t use headings for highlighting any word or sentence. 



5. What will happen if too many headings are used in a page?

A. Web crawlers will SPAM your page.



6. How many headings can be in page?

A. There is no limit but depends on content in page. 



12. Paragraphs & Blockquotes 



    - <p> element is used for paragraph.

    - <blockquote> is used for block quote, which is a summary for content in page. [SEO]



FAQ’s:

1. What is the role of block quote? 

A. It summarizes the content in page and makes it SEO friendly. 



2. How to set first line indent in a paragraph or blockquote?  

A. By using CSS “text-indent” attribute. 



Syntax:

           p, blockquote {

                   text-indent: 100px;

           }



3. How to set space between line, words and chars?

A. By using CSS attributes

               

       a) letter-spacing

       b) word-spacing

       c) line-height 



Syntax:

       p, blockquote {

          letter-spacing : 2px;

          word-spacing: 5px;

          line-height: 40px;

       }



4. How to set a drop cap?

A.

       a) Access the first letter by using CSS  “::first-letter”  class



                       p :: first-letter {  }



       b) Set font size and family etc.

       c) Set CSS “float” to left. 

       d) Control the line height. [ space between lines ]



Syntax:

       p :: first-letter {

               font-size : 40px;

               font-family: Arial;

               float: left;

               line-height: 40px;

         }
