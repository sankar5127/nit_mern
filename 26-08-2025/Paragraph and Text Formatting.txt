6. How to use ellipsis in paragraph?

A. By using CSS attributes

       a) overflow

       b) text-overflow

       c) white-space 



Syntax:

       p {

           width : 400px;

           border: 1px solid gray;

           padding: 5px;

           white-space: nowrap;

           overflow: hidden;

           text-overflow: ellipsis;

       }

8. How to display paragraph in multiple columns?

A. You can use “display” with “grid” style.

   You can use CSS “columns”.



   a) columns

   b) column-gap

   c) column-width

   d) column-rule



Syntax:

   .container {

           columns : 3;

           column-gap: 20px;

           column-rule : 1px dotted gray;

    }

Text Formatting in HTML



1. <font>            It configure face, size and color for text. 

                       a) face    : It refers font family name.

                       b) size    : It refers to level number from 1 to 7.

                       c) color    : It refers to color name, code etc.

Syntax:

       <font face=“Arial” size=“7” color=“#f00”>  Text </font>



FAQ’s: 

1. What is “7” refers to in font size?

A. HTML font size can be configures in 7 levels, browser uses levels as

       

           1 x-small

           2 small

           3 normal

           4 large

           5 x-large

           6 xx-large

           7 xxx-large 



2. What is default font size?

A. It is “3”.



3. What are web safe fonts?

A. Web safe refers to fonts that are available across all devices. 

   The default web safe fonts are

       a) serif

       b) sans-serif

       c) monospace 



Note: Never use <font> in HTML. It is not recommended for modern web design.
