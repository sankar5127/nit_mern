CSS Sticky Position:

   - CSS provides various position attributes        a) static

       b) absolute

       c) relative

       d) fixed

       e) sticky

   - Sticky allows the to keep the element according to normal flow of document, but it sticks

     at specific position top, right, bottom or left. So that it locks scrolling and allows to 

     scroll other content in backdrop. 



Syntax:

        dt {

             position : sticky;

             top: 0px;

         }

2. Ordered List



       <ol>            ordered list

       <li>            list item



   - It is auto numbering for a list of items.

   - Numbering updates automatically when you add or remove item.



Syntax:

           <ol>    

               <li> Item -1 </li>

               <li> Item -2 </li>

           </ol>



    - You can set numbering style by using “type” attribute with “1, A, a, i, I”.

    

           <ol type=“a”>



    - You can set start level, that defines from where the numbering has to start. 



            <ol  type=“1” start=“101”>          start => is level number

       

            <ol type=“a” start=“5”> 



     - You can reverse the numbering by using “reversed” attribute.



             <ol type=“1” start=“2” reversed>



     - You can create nested list, but make sure that the child list is inside <li> element.



Ex:
