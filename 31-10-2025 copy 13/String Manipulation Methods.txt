4. startsWith()

       - It returns true if string starts with specified char(s).

   

5. endsWith()

       - It returns true if string ends with specified char(s).



Syntax:

       var str = “welcome”;

       str.startsWith(“w”);       // true

       str.startsWith(“W”);     // false 

       str.endsWith(“me”);     // true

6. indexOf()

           - It returns the index of specified character in a string. 

           - If character not found then it returns “-1”.

           - It finds and returns the first occurrence index number. 



7. lastIndexOf()

           - It returns the last occurrence index number of specified char in a string. 



   Syntax:

         var str = “welcome”;

         str.indexOf(“e”);            // 1

         str.lastIndexOf(“e”);          // 6

         str.indexOf(“a”);             // -1

8. slice()                : It is used to slice and return the chars between specified index.

                             It uses a start & end index. 

                             End Index must be a value greater than start index.



Syntax:

         str.slice(startIndex, endIndex);



9. substr()            : It returns specified number of chars from given index. 

                             It uses count of chars that must be above zero. 



Syntax:

           str.substr(startIndex, countOfChars)



10. substring()     : It returns the chars between specified index. 

                              It can read bi-directional.

                              End index reference can be to start or end. 



Syntax:

           str.substring(startIndex, endReference)



Note: If only the start index is defined for all methods, then they can read upto end. 



           str.slice(startIndex)

           str.substr(startIndex)

           str.substring(startIndex)

       var email = prompt(“enter email”);          //     John@gmail.com, david123@outlook.com

    var domain = email.substring(atPos+1);

    document.write(`Id=${id}<br>Domain=${domain}`);

</script>

12. split()

           - It splits a string at specified delimiter and returns an array of strings. 



Syntax:

       var str = “A-B-C”;

       str.split(‘-‘);     [A, B, C]    0=A, 1=B, 2=C



Ex:

<script>

    var contacts = "John-9888772277, David-9000772200";

    var [john, david] = contacts.split(','); // [0,1]

    document.write("John Mobile " + john.split('-')[1]);

</script>
